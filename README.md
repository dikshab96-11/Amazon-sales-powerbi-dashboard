# 📊 Amazon Sales Data Analysis Dashboard (Power BI)

## 🔍 Project Overview
This project presents an end-to-end analysis of Amazon e-commerce sales data using Power BI. The goal is to derive actionable insights related to sales performance, profitability, and regional trends.

---

## 🎯 Objectives
- Analyze overall sales, profit, and shipping cost
- Identify top-performing countries and cities
- Understand sales distribution across markets and categories
- Enable interactive data exploration through dashboards

---

## 📁 Dataset
- Source: Amazon sales dataset (sample dataset)
- Records: 10,000+ transactions
- Features include:
  - Order Date
  - Ship Mode
  - Country, City
  - Sales, Profit, Shipping Cost
  - Category

---

## 🛠 Tools & Technologies
- Power BI
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)

---

## 📊 Dashboard Features
- KPI Cards: Total Sales, Quantity, Profit, Shipping Cost
- Sales by Country (Top contributing regions)
- Sales by City (Detailed breakdown)
- Sales by Ship Mode
- Sales by Market (APAC, EU, US, etc.)
- Interactive Filters (Category, Market)

## 📈 Key Insights
- APAC region contributes ~96% of total sales
- Top-performing cities include Jakarta, Manila, and Bangkok
- Standard Class shipping dominates sales volume
- Sales are highly concentrated in specific regions
